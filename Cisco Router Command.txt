Access Control List-ACL:
Standard ACL:
Router(config)#access-list 1 deny host 192.168.1.1
Router(config)#do show access-lists
Router(config)#int fa4/0
Router(config-if)#ip access-group 1 in 

Remove:
Router(config)#no access-list 1

Extended List:
Router(config)#access-list 100 deny icmp 192.168.1.0 host 192.168.3.1 echo
Router(config)#access-list 100 permit icmp any any echo
Router(config)#int fa0/1
Router(config-if)#ip access-group 100 in 

Network Address Translations - NAT 
Router(config)#access-list 1 permit 192.168.1.0 0.0.0.255
Router(config)#ip nat inside source list 1 int fa4/0 overload
Router(config)#do show ip nat Translations

Using pool:
Router(config)#ip nat pool pool_1 209.165.200.1 209.165.200.4 netmask 255.255.255.0
Router(config)#ip nat inside source list 1 pool pool_1 overload

Router(config)#int fa0/0
Router(config-if)#ip nat inside 
Router(config)#int fa4/0
Router(config-if)#ip nat outside
